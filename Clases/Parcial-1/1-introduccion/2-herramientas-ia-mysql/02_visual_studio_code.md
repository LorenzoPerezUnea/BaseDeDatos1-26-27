# Visual Studio Code

Visual Studio Code, conocido habitualmente como VS Code, es uno de los editores de código más utilizados del mundo.

Aunque muchas personas lo confunden con un entorno de desarrollo completo (IDE), en realidad nació como un editor ligero y altamente extensible.

Su filosofía consiste en proporcionar una base sencilla sobre la que cada usuario instala únicamente las funcionalidades que necesita.

### ¿Por qué utilizaremos VS Code?

Durante este curso escribiremos mucho más que consultas SQL.

También redactaremos documentación, analizaremos archivos JSON, configuraremos Docker, consultaremos scripts, visualizaremos diagramas Mermaid y trabajaremos con Markdown.

VS Code permite realizar todas estas tareas desde una única aplicación.

### Ventajas principales

* Gratuito.
* Multiplataforma.
* Gran rendimiento.
* Miles de extensiones.
* Excelente soporte para Git.
* Integración con Docker.
* Excelente soporte para SQL.

### Extensiones recomendadas

Durante el curso instalaremos algunas extensiones muy útiles:

* Markdown All in One
* Mermaid Preview
* Docker
* SQLTools
* GitLens

No es necesario instalarlas todas el primer día, pero sí es recomendable hacerlo progresivamente.

### Terminal integrada

Una de las características más útiles de VS Code es su terminal integrada.

Gracias a ella podemos ejecutar comandos sin abandonar el editor.

```bash
docker ps
```

```bash
git status
```

```bash
mysql --version
```

Esto evita cambiar continuamente entre diferentes aplicaciones.

### Ideas clave

* VS Code es un editor profesional.
* Puede ampliarse mediante extensiones.
* Integra herramientas de desarrollo modernas.
* Será el centro de trabajo durante el semestre.
