# 05. Información, conocimiento y toma de decisiones

### El siguiente nivel

Una vez que los datos se convierten en información aparece una nueva pregunta.

¿Qué hacemos con ella?

La respuesta es sencilla.

Tomar decisiones.

### Un ejemplo empresarial

Supongamos que observamos el siguiente informe.

```text
Producto A: 120 ventas

Producto B: 15 ventas

Producto C: 8 ventas
```

Eso es información.

Ahora un gerente observa esos resultados y concluye:

* El Producto A tiene alta demanda.
* Debemos aumentar el inventario.
* Debemos revisar la estrategia comercial de los otros productos.

Acaba de generarse conocimiento.

### La importancia para las organizaciones

Las empresas no invierten en bases de datos porque quieran almacenar registros.

Invierten porque desean tomar mejores decisiones.

Las bases de datos permiten responder preguntas críticas.

* ¿Qué productos venden más?
* ¿Qué clientes generan más ingresos?
* ¿Qué sucursal es más rentable?
* ¿Qué empleados obtienen mejores resultados?

Responder correctamente a estas preguntas puede significar la diferencia entre el éxito y el fracaso de una organización.

### Idea clave

El valor real de una Base de Datos no reside en almacenar información sino en permitir tomar decisiones fundamentadas.
