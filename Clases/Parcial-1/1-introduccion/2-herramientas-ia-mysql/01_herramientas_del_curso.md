# Herramientas del curso

La informática es una disciplina eminentemente práctica. No basta con conocer la teoría; es necesario experimentar, cometer errores, corregirlos y volver a intentarlo.

Por esta razón, a lo largo del semestre utilizaremos un conjunto reducido de herramientas profesionales. Son las mismas, o muy similares, a las empleadas diariamente por desarrolladores, administradores de bases de datos e ingenieros de software.

El objetivo no es aprender muchas herramientas diferentes, sino dominar unas pocas que permitan trabajar de forma eficiente.

### ¿Por qué necesitamos varias herramientas?

Podría parecer que una única aplicación sería suficiente para crear bases de datos, escribir consultas SQL y desarrollar aplicaciones.

Sin embargo, en proyectos reales cada herramienta está especializada en resolver un problema concreto.

Podemos compararlo con un taller mecánico.

Un mecánico utiliza llaves inglesas, destornilladores, elevadores y herramientas de diagnóstico. Todas forman parte del mismo trabajo, pero ninguna sirve para todo.

En informática ocurre exactamente igual.

### Herramientas que utilizaremos

| Herramienta        | Función                                  |
| -------------------- | ------------------------------------------- |
| Visual Studio Code | Editor de código                         |
| MySQL Workbench    | Administración de MySQL                  |
| Docker             | Ejecución de contenedores                |
| GitHub             | Material del curso y control de versiones |
| ChatGPT            | Asistente de aprendizaje                  |

Cada una aparecerá constantemente durante el semestre.

### ¿Por qué usar herramientas profesionales?

Existen aplicaciones mucho más sencillas para realizar consultas SQL.

Entonces, ¿por qué no utilizarlas?

Porque el objetivo de la universidad no consiste únicamente en aprender una asignatura, sino en adquirir competencias profesionales.

Aprender desde el principio herramientas ampliamente utilizadas facilita posteriormente la incorporación al mundo laboral.

Además, muchas de ellas son gratuitas y multiplataforma, lo que permite continuar practicando desde cualquier ordenador.

### Caso práctico del curso

Durante todo el semestre trabajaremos sobre una empresa comercial ficticia.

Comenzaremos con una base de datos muy sencilla y, poco a poco, irá creciendo hasta incorporar clientes, productos, pedidos, facturas, empleados, almacenes, proveedores y muchos otros elementos.

Todas las herramientas estudiadas en esta clase se utilizarán para desarrollar dicho caso práctico.

### Ideas clave

* Una herramienta resuelve un problema específico.
* No existe una aplicación que haga todo perfectamente.
* Utilizaremos herramientas profesionales reales.
* Todas las prácticas compartirán el mismo entorno de trabajo.
* El caso práctico evolucionará durante todo el curso.
