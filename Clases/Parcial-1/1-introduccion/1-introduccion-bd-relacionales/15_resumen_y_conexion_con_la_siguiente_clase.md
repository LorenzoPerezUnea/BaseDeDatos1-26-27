# 15. Resumen y conexión con la siguiente clase

Con esta primera clase hemos sentado las bases conceptuales sobre las que construiremos todo el curso.

Hemos comprendido que una Base de Datos no surge simplemente para almacenar información, sino para resolver problemas reales relacionados con la organización, el mantenimiento y el acceso a los datos.

También hemos visto cómo los sistemas basados en archivos presentaban limitaciones importantes y cómo el Modelo Relacional ofreció una solución mucho más flexible y robusta.

Además, hemos conocido a Edgar F. Codd, las ideas que dieron origen al Modelo Relacional y el papel de los Sistemas Gestores de Bases de Datos como software encargado de administrar la información.

Finalmente, hemos presentado el caso de estudio que desarrollaremos durante todo el semestre: el sistema de información de ​**Comercial Nova**​.

### Lo que hemos aprendido

Al finalizar esta sesión deberías ser capaz de responder preguntas como:

* ¿Qué diferencia existe entre un dato, información y conocimiento?
* ¿Por qué los sistemas basados en archivos dejaron de ser suficientes?
* ¿Qué es una Base de Datos?
* ¿Qué objetivos persigue una Base de Datos moderna?
* ¿Quién fue Edgar F. Codd?
* ¿Qué es un Sistema Gestor de Bases de Datos?
* ¿Cuál será el proyecto que desarrollaremos durante el semestre?

Si puedes responder con seguridad a estas preguntas, dispones de una base sólida para continuar.

### Conexión con la siguiente clase

En la próxima sesión dejaremos de centrarnos en la historia de las Bases de Datos para estudiar el entorno de trabajo que utilizaremos durante todo el semestre.

Conoceremos las principales herramientas del curso, aprenderemos cómo la Inteligencia Artificial puede utilizarse de forma responsable como apoyo al aprendizaje y entenderemos la arquitectura cliente-servidor, imprescindible para comprender cómo una aplicación se comunica con un Sistema Gestor de Bases de Datos.

Esta arquitectura será el puente entre la teoría aprendida en esta primera clase y la implementación práctica que comenzaremos a desarrollar en las siguientes sesiones.
