# 04. Datos vs Información

### ¿Son lo mismo?

En el lenguaje cotidiano solemos utilizar ambas palabras como sinónimos.

Sin embargo, representan conceptos distintos.

Un dato es un hecho aislado.

La información surge cuando varios datos se organizan y adquieren significado.

### Un ejemplo sencillo

Observemos los siguientes datos.

```text
Ana Ruiz

Portátil

850 €

3 unidades
```

Por separado son simples datos.

Ahora observémoslos juntos.

```text
Cliente: Ana Ruiz

Producto: Portátil

Cantidad: 3

Precio unitario: 850 €
```

Ya no observamos datos aislados.

Observamos una venta.

Ha aparecido información.

### El poder de las relaciones

La información surge cuando los datos se relacionan.

Esta idea es tan importante que explica el éxito del modelo relacional.

Las bases de datos modernas no se limitan a almacenar datos.

Almacenan relaciones entre datos.

### Ejemplo empresarial

Una lista de precios es un conjunto de datos.

Un informe de ventas es información.

Un análisis de tendencias comerciales es conocimiento.

Cada nivel aporta más valor que el anterior.

### Idea clave

La información no aparece por acumulación de datos.

Aparece cuando los datos se organizan y adquieren significado.
